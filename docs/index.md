# Graphina

  <img src="../logo.png" alt="Graphina Logo" width="300" />
</p>

**Graphina** is a high-performance graph data science library for Rust.
It provides a robust collection of graph algorithms, data structures, and analytics tools designed for efficiency and ease of use.

## Features

*   **Core Graph Types**: Flexible `Graph` (undirected) and `Digraph` (directed) implementations.
*   **Performance**: Optimized for speed and memory usage, with parallel implementations for heavy workloads.
*   **Algorithms**: Comprehensive suite including Centrality, Community Detection, Path Finding, and more.
*   **Interoperability**: Python bindings available via `pygraphina`.
